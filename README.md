# Rasterização de Triângulos 3D com Scanline em Python

Este projeto implementa um pipeline básico de visualização 3D em Python, 
incluindo transformação de vértices, projeção em NDC e rasterização de triângulos usando o algoritmo clássico de scanline.

##  Objetivo

O objetivo deste projeto é estudar e implementar conceitos fundamentais
de computação gráfica, como transformação de coordenadas, projeção
perspectiva e rasterização de primitivas 3D, sem o uso de APIs gráficas
de alto nível.

## ⚙️ Funcionalidades

- Representação de sólidos 3D por vértices e faces trianguladas
- Transformação para espaço da câmera
- Projeção perspectiva
- Rasterização de triângulos via algoritmo de scanline
- Colorização de objetos individualmente
- Visualização dos planos near e far

##  Contexto Acadêmico

Projeto desenvolvido como parte da disciplina de Computação Gráfica,
com foco em compreender o pipeline gráfico em baixo nível.
